# Cara Menambahkan Image Wedison Hero

## Langkah-langkah:

1. **Siapkan Image**
   - Format: JPG, PNG, atau WebP
   - Ukuran: 1920x1080px atau lebih besar (16:9 aspect ratio)
   - File size: Optimize ke <500KB untuk loading cepat
   - Konten: Dark blue maxi-scooter di winter landscape

2. **Rename File**
   - Nama file harus: `wedison-hero.jpg` (atau .png/.webp)
   - Pastikan nama file sesuai dengan path di HeroSlider.tsx

3. **Place Image**
   - Copy file image ke folder: `/public/images/brand/`
   - Path lengkap: `/public/images/brand/wedison-hero.jpg`

4. **Verify**
   - Image akan otomatis muncul di hero slider slide pertama
   - Jika tidak muncul, cek:
     - Nama file harus exact: `wedison-hero.jpg`
     - File harus di folder `/public/images/brand/`
     - Refresh browser (Ctrl+F5 atau Cmd+Shift+R)

## Image Description (Reference):
- Dark blue maxi-scooter (Wedison electric motorcycle)
- Winter landscape dengan snow-covered mountains
- Dense forest of snow-laden evergreen trees
- Serene, professional product photography
- Road dengan snowbanks di kedua sisi

## Tips:
- Gunakan image optimization tool untuk mengurangi file size
- Pastikan image quality tetap tinggi untuk hero section
- Test di berbagai device untuk memastikan responsive
