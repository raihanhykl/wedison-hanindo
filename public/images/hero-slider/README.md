# Hero Slider Images

Place your hero slider images here for the landing page.

## Image Files Required

### 1. Sewa Slide
- **File Name**: `hero-sewa.jpg` (or `.png`, `.webp`)
- **Description**: Image for rental/promo slide
- **Recommended Size**: 1920x1080px (16:9 aspect ratio)
- **Content**: Scooter with rental theme, flexible/promo concept

### 2. Beli/Subsidi Slide
- **File Name**: `hero-beli-subsidi.jpg` (or `.png`, `.webp`)
- **Description**: Image for purchase/subsidy promo slide
- **Recommended Size**: 1920x1080px (16:9 aspect ratio)
- **Content**: Electric motorcycle with subsidy/purchase promo theme

### 3. Voucher Charging Slide
- **File Name**: `hero-voucher-charging.jpg` (or `.png`, `.webp`)
- **Description**: Image for charging voucher promo slide
- **Recommended Size**: 1920x1080px (16:9 aspect ratio)
- **Content**: Charging station or motorcycle charging with voucher theme

### 4. Models Slide
- **File Name**: `hero-models.jpg` (or `.png`, `.webp`)
- **Description**: Image for explore models slide
- **Recommended Size**: 1920x1080px (16:9 aspect ratio)
- **Content**: Multiple models or single hero model showcasing all options

## Image Specifications

- **Format**: JPG, PNG, or WebP (WebP recommended for best performance)
- **Recommended Size**: 1920x1080px (16:9 aspect ratio) or larger
- **File Size**: Optimize to <500KB per image for fast loading
- **Quality**: High-quality, professional photography
- **Content**: Should work well with text overlay (darker areas for text readability)

## How to Add Images

1. **Upload your images** to this folder (`public/images/hero-slider/`)
2. **Name them exactly** as listed above (e.g., `hero-sewa.jpg`)
3. **Optimize images** using tools like:
   - [TinyPNG](https://tinypng.com/) - for PNG/JPG
   - [Squoosh](https://squoosh.app/) - for WebP conversion
4. The code will automatically use these images once uploaded
